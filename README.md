Contact Form with Admin Panel
Project Description

This is a simple Contact Form project made using PHP, HTML, and MySQL.
Users can send messages using a contact form.
Admin can login and view submitted messages.

Technologies Used

PHP

HTML

MySQL

XAMPP (Local Server)

Features

User contact form

Save data in MySQL database

Admin login

Admin can view messages

Import database.sql

http://localhost/contact-form
